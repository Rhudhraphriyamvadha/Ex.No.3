# 1. STRAIGHTFORWARD PROMPTS

## Definition

Straightforward prompting directly asks the AI for a simple answer without additional context.

## Example Prompt 1

“Define Machine Learning in one sentence.”

### AI Response

“Machine Learning is a branch of artificial intelligence that enables computers to learn patterns from data and make decisions without explicit programming.”

---

## Example Prompt 2

“List three advantages of online learning.”

### AI Response

1. Flexible learning schedule
2. Easy access to educational resources
3. Cost-effective learning

---

## Observation

* Fast response generation
* Simple and direct answers
* Best for quick information retrieval

---

# 2. TABULAR FORMAT PROMPTING

## Definition

Tabular prompting requests information in a structured table format for easy comparison and understanding.

## Example Prompt

“Provide a table comparing supervised and unsupervised learning.”

### AI Response

| Feature      | Supervised Learning | Unsupervised Learning |
| ------------ | ------------------- | --------------------- |
| Data Type    | Labeled Data        | Unlabeled Data        |
| Goal         | Predict Output      | Discover Patterns     |
| Examples     | Classification      | Clustering            |
| Applications | Spam Detection      | Customer Segmentation |

---

## Observation

* Information becomes visually organized
* Easy comparison of concepts
* Useful for academic learning

---

# 3. PRECEDING QUESTION PROMPTING

## Definition

Preceding Question prompting asks a leading question before the main instruction to guide AI reasoning.

## Example Prompt

“Why is cybersecurity important? Explain how encryption protects user data.”

### AI Response

Cybersecurity is important because it protects systems, networks, and personal data from cyber attacks. Encryption secures user data by converting readable information into coded text that can only be accessed using a decryption key.

---

## Observation

* Produces more detailed explanations
* Improves logical reasoning
* Generates context-aware responses

---

# 4. MISSING WORD PROMPTING

## Definition

Missing Word prompting asks the AI to complete incomplete sentences.

## Example Prompt 1

“The brain of the computer is called the ____.”

### AI Response

“CPU”

---

## Example Prompt 2

“In artificial intelligence, machines learn from ____.”


## Observation

* Useful for quizzes and assessments
* Improves student engagement
* Best for interactive learning systems
